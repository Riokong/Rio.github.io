**Project Overview**  
Full-stack web app to manage family members and track visited countries.

**Highlights**
- Built with **Node.js**, **Express**, **EJS** templating.
- **PostgreSQL** schema with relational constraints to ensure integrity and prevent duplicates.
- Interactive **SVG world map**; dynamically color-codes visited countries per user.

**Tech**
Node.js, Express, EJS, PostgreSQL, SVG, CSS
